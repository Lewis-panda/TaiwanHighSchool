# 物理科 學習內容（官方條目）

> [!info] 來源與代碼
> 教育部《十二年國民基本教育課程綱要—自然科學領域》。物理代碼格式：`P + 主題 + -Ⅴx- + 流水號`。
> - 主題：B 能量的形式轉換及流動、E 物質系統、K 自然界的現象與交互作用、M 科學科技社會及人文、N 資源與永續。
> - **Ⅴc = 高中必修**；**Ⅴa = 高中加深加廣選修**。（「Ⅴ」是羅馬數字，代表第五學習階段。）
>
> 課綱的學習內容是依**跨科主題**編排，不是依課本章節；同一章往往對應分散的數個條目。下方先列原文，再給「章 ↔ 條目」對照。

## 二、選修物理（Ⅴa）— 聚焦力學

### 物質系統（E）— 力與運動 Eb
| 代碼 | 學習內容 |
|---|---|
| PEa-Ⅴa-1 | 測量都會有不確定度，源自被測物、儀器特性，並受測量者與環境影響；有國際公定的計算方法。 |
| PEb-Ⅴa-1 | 質點在平面上運動，其位移、速度、加速度有兩個獨立的分量。 |
| PEb-Ⅴa-2 | 直線等加速運動（例如自由落體），其位移、速度、加速度及時間的數學關係。 |
| PEb-Ⅴa-3 | 二質點在同一直線上運動，其相對速度為二質點速度之差。 |
| PEb-Ⅴa-4 | 簡諧運動為一週期性運動，其位移和速度可用時間的正弦或餘弦函數表示。 |
| PEb-Ⅴa-5 | 質點在平面運動時位移速度加速度有兩分量，用向量表示，例如拋體運動，軌跡是拋物線。 |
| PEb-Ⅴa-6 | 質點作等速圓周運動時速率與角速度不變，但有向心加速度，故速度方向會改變。 |
| PEb-Ⅴa-7 | 力是向量，可以分解和合成。 |
| PEb-Ⅴa-8 | 牛頓三大運動定律：慣性定律、運動定律、作用與反作用定律。 |
| PEb-Ⅴa-9 | 牛頓第二運動定律的應用，例如簡諧運動與等速圓周運動。 |
| PEb-Ⅴa-10 | 質點動量等於質量乘以速度，其時間變化率等於所受作用力；衝量等於動量的變化。 |
| PEb-Ⅴa-11 | 質點系統動量對時間的變化率等於外力總和；若外力總和為零，則系統動量守恆。 |
| PEb-Ⅴa-12 | 質點系統質心的定義。 |
| PEb-Ⅴa-13 | 質心速度、質心加速度及系統總動量與所受外力的關係。 |
| PEb-Ⅴa-14 | 質點的角動量等於位置向量與動量的向量外積，其時間變化率等於所受力矩。 |
| PEb-Ⅴa-15 | 許多生活與工程問題可用牛頓三大運動定律解釋或計算，例如靜力平衡、摩擦力、一維碰撞問題。 |

> 「功、動能、功率、位能、力學能守恆」屬選修力學核心，課綱對應於能量主題（B）之選修條目；本庫 `選物II-2 功與能量` 章涵蓋之。撰寫時若需精確條碼，請再查領綱能量主題 Ⅴa 條目，勿臆造代碼。
