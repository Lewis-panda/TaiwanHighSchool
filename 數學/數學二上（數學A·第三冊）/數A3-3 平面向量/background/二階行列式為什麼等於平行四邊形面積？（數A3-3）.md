# 二階行列式為什麼等於平行四邊形面積？

> [!question] 課本沒講清楚的地方
> 課本直接說：以 $\vec a=(a_1,a_2)$、$\vec b=(b_1,b_2)$ 為鄰邊的平行四邊形，面積等於 $|a_1b_2-a_2b_1|$。可是面積不是「底乘高」嗎？這個 $a_1b_2-a_2b_1$ 是哪冒出來的？而且為什麼還會有正負號？

## 課本怎麼說

課本通常只給結論並舉一個例子驗證，沒有推導；學生只能背「左上乘右下減右上乘左下，取絕對值」。這讓行列式變成一個咒語，也不知道那個正負號代表什麼。

## 更深一層

### 法一：面積 ＝ 底 × 高，把高用三角函數寫出來

平行四邊形以 $\vec a$ 為底，底長 $=|\vec a|$。高 $=$ $\vec b$ 在「垂直於 $\vec a$」方向上的分量 $=|\vec b|\sin\theta$（$\theta$ 是 $\vec a$、$\vec b$ 的夾角）。所以

$$A=|\vec a||\vec b|\sin\theta.$$

現在把 $\sin\theta$ 換成坐標。我們已知內積給了 $\cos\theta$：$\vec a\cdot\vec b=|\vec a||\vec b|\cos\theta=a_1b_1+a_2b_2$。用 $\sin^2\theta=1-\cos^2\theta$：

$$A^2=|\vec a|^2|\vec b|^2\sin^2\theta=|\vec a|^2|\vec b|^2-(|\vec a||\vec b|\cos\theta)^2=|\vec a|^2|\vec b|^2-(\vec a\cdot\vec b)^2.$$

把每一項用坐標代入（$|\vec a|^2=a_1^2+a_2^2$ 等）：

$$A^2=(a_1^2+a_2^2)(b_1^2+b_2^2)-(a_1b_1+a_2b_2)^2.$$

把它硬乘開（這一步是關鍵）：

$$=a_1^2b_1^2+a_1^2b_2^2+a_2^2b_1^2+a_2^2b_2^2-(a_1^2b_1^2+2a_1b_1a_2b_2+a_2^2b_2^2).$$

$a_1^2b_1^2$ 與 $a_2^2b_2^2$ 各自消掉，剩下：

$$A^2=a_1^2b_2^2-2a_1b_2a_2b_1+a_2^2b_1^2=(a_1b_2-a_2b_1)^2.$$

開根號（面積非負取絕對值）：

$$\boxed{\;A=|a_1b_2-a_2b_1|\;}$$

那個看似突兀的 $a_1b_2-a_2b_1$，原來是「**長度平方乘積減去內積平方**」自然收斂出來的完全平方。這個恆等式

$$(a_1^2+a_2^2)(b_1^2+b_2^2)=(a_1b_1+a_2b_2)^2+(a_1b_2-a_2b_1)^2$$

叫**拉格朗日恆等式**，本身就漂亮——「平方和的乘積 ＝ 內積平方 ＋ 行列式平方」。它也立刻給出柯西不等式（右邊第二項 $\ge0$，所以內積平方 $\le$ 長度平方乘積）。

### 法二：切割重組（純幾何，給不想用三角的班級）

把 $\vec a$、$\vec b$ 圍成的平行四邊形放進一個 $(a_1+b_1)\times$ 高度的大矩形裡，用「大矩形面積，減掉周圍幾塊三角形與小矩形」的方式湊。逐塊扣下來，化簡後也會得到 $a_1b_2-a_2b_1$。這個做法不需 $\sin\theta$，但代數較瑣碎；建議程度好的班用法一、想看幾何的班看法二的圖示即可。

### 正負號代表什麼：定向

行列式 $a_1b_2-a_2b_1$ **本身可正可負**，取絕對值才是面積。那個正負號編碼了 $\vec a$ 到 $\vec b$ 的**旋轉方向（定向 orientation）**：

- $a_1b_2-a_2b_1>0$：從 $\vec a$ 轉到 $\vec b$ 是**逆時針**。
- $a_1b_2-a_2b_1<0$：從 $\vec a$ 轉到 $\vec b$ 是**順時針**。
- $=0$：$\vec a$、$\vec b$ **共線（平行）**，平行四邊形被壓扁成一條線，面積為 $0$——這正是 3-2 的平行判定！

> [!warning] 常見迷思
> - 別漏取絕對值。算出 $-11$ 不代表面積是負的，它只是說頂點排列是順時針，面積仍是 $11$。
> - 行列式的兩列要放「**從同一頂點出發的兩個邊向量**」，不是三個頂點的坐標。
> - 「行列式為 0」對應「平行 / 面積為 0」，不是「垂直」。垂直是**內積**為 0。兩件事剛好相反，務必分清。

## 回到課堂
- **要不要推導**：法一（底乘高 + $\sin^2=1-\cos^2$）建議完整推一次，順便把拉格朗日恆等式秀出來，再指出「它右邊平方非負 → 柯西不等式」，能把 3-4 三個主題（面積、平行、柯西）一次串起來。
- **一句話講法**：面積 $=|\vec a||\vec b|\sin\theta$，把 $\sin$ 用坐標寫出來，自然湊成 $(a_1b_2-a_2b_1)^2$。
- **正負號的意義**：逆時針為正、順時針為負、共線為零（壓扁）。
- **預期學生會追問**：「那行列式為 0 為什麼是平行？」——面積為 0 表示平行四邊形被壓成一條線，兩鄰邊只能在同一直線上，即平行。
