# 空間中怎麼判斷兩向量垂直、兩向量平行？

> [!question] 課本沒講清楚的地方
> 課本給「$\vec a\cdot\vec b=0$ 就垂直」「分量成比例就平行」兩條規則，學生卻常常**用反**（拿內積去判平行、拿比例去判垂直），也說不出為什麼是這兩條。

### 一句話：垂直看內積、平行看比例（或外積）
| 想判斷 | 用什麼 | 條件 | 為什麼 |
|---|---|---|---|
| **垂直** $\vec a\perp\vec b$ | 內積 | $\vec a\cdot\vec b=0$ | $\vec a\cdot\vec b=|\vec a||\vec b|\cos\theta$，垂直時 $\cos90^\circ=0$ |
| **平行** $\vec a\parallel\vec b$ | 比例 / 外積 | $\vec b=t\vec a$（分量成比例）；或 $\vec a\times\vec b=\vec0$ | 平行時 $\sin\theta=0$，外積長度為 0 |

### 細一點：為什麼垂直恰好是「內積 = 0」？
內積定義 $\vec a\cdot\vec b=|\vec a||\vec b|\cos\theta$。兩個非零向量的長度都正，所以
$$\vec a\cdot\vec b=0\iff\cos\theta=0\iff\theta=90^\circ.$$
*類比*：內積在量「$\vec b$ 有多少分量是沿著 $\vec a$ 方向的」。如果 $\vec b$ 完全垂直於 $\vec a$，它沒有任何分量落在 $\vec a$ 方向上，這個「沿 $\vec a$ 的分量」就是 0——內積自然是 0。

**用坐標檢查**：$\vec a=(1,2,2)$、$\vec b=(2,-2,1)$，$\vec a\cdot\vec b=2-4+2=0$，垂直 ✓。不用算角度，一眼看出。

### 細一點：平行為什麼用「比例」？要不要也記外積？
平行（共線）表示一個是另一個的倍數：$\vec b=t\vec a$，即
$$\frac{b_1}{a_1}=\frac{b_2}{a_2}=\frac{b_3}{a_3}\ (=t).$$
*例*：$\vec a=(2,-3,k)$ 與 $\vec b=(-4,6,10)$，由 $\dfrac{-4}{2}=\dfrac{6}{-3}=-2$ 知比例 $-2$，再用第三分量 $10=-2k$ 解 $k=-5$。

也可以用**外積**：$\vec a\parallel\vec b\iff\vec a\times\vec b=\vec0$（外積長度 $=|\vec a||\vec b|\sin\theta$，平行時 $\sin\theta=0$）。比例法在「有未知數要解」時方便，外積法在「順手已算外積」時方便。

> [!warning] 常見迷思
> - **拿內積判平行**：有人看到 $\vec a\cdot\vec b$ 很大就說平行——錯。內積大只代表夾角小，不一定平行。平行的鐵證是**比例相同**或**外積為零**。
> - **比例式裡有 0 分量**：若某分量為 0（如 $\vec a=(2,0,3)$），「除以 0」要小心，改寫成 $\vec b=t\vec a$ 逐項對應比較安全：要平行，$\vec b$ 對應那一項也得是 0。
> - **零向量**：$\vec0$ 與任何向量都「平行也垂直」，討論時通常排除零向量。

### 回到課堂
- 一句話講法：**「垂直 → 內積 0；平行 → 成比例（或外積為零）。一個管角度為 90°，一個管角度為 0°。」**
- 好用對照：$\cos\theta$（內積裡）在 90° 為 0 ⟹ 管垂直；$\sin\theta$（外積裡）在 0° 為 0 ⟹ 管平行。把兩個三角函數的零點記住，兩條規則就不會用反。
- 預期學生會追問：「那夾角不是 0 也不是 90 呢？」答：用 $\cos\theta=\dfrac{\vec a\cdot\vec b}{|\vec a||\vec b|}$ 算出來，這是內積最常見的用途。
