# 等加速度三公式是怎麼來的、要不要硬背？

> [!question] 課本沒講清楚的地方
> 課本一口氣丟出三條等加速度公式 $v=v_0+at$、$\Delta x=v_0t+\tfrac12at^2$、$v^2=v_0^2+2a\Delta x$，要學生背起來套。可是這三條**從哪來**？彼此什麼關係？真的要硬背嗎？背錯一個符號整題就崩，有沒有更穩的辦法？

## 一句話講它怎麼來的
三條公式**全部出自同一張 $v\text{–}t$ 圖**：等加速度時 $v\text{–}t$ 圖是一條直線，**斜率＝加速度**給你第一條、**下方面積＝位移**給你第二條、把第一條代進第二條消去時間就得第三條。理解了這條主線，幾乎不用背。

## 細一點

### 出發點只有一張圖
等加速度 $a$ 為定值 ⇒ $v\text{–}t$ 圖是**斜率 $a$、截距 $v_0$ 的直線**。

```
 v
 │          ╱  ← 斜率 = a
 │        ╱
 v ──────●
 │      ╱│
 │    ╱  │
v0 ●╱    │     面積（梯形）= 位移
 │ │     │
 └─┴─────┴──► t
   0     t
```

**第一條（斜率法）**：直線斜率固定為 $a$，速度從 $v_0$ 開始、每秒增加 $a$，到時刻 $t$ 增加了 $at$：
$$v = v_0 + at.\tag{1}$$
這其實就是加速度定義 $a=\dfrac{v-v_0}{t}$ 的移項，沒有任何新東西。

**第二條（面積法）**：位移＝圖下方梯形面積。梯形上底 $v_0$、下底 $v$、高 $t$：
$$\Delta x = \frac{v_0+v}{2}\,t.\tag{$\ast$}$$
把 $(\ast)$ 裡的 $v$ 用 $(1)$ 換掉：
$$\Delta x = \frac{v_0+(v_0+at)}{2}\,t = v_0 t + \tfrac12 at^2.\tag{2}$$
也可以直接把梯形拆成「矩形 $v_0 t$ ＋ 三角形 $\tfrac12(at)t$」，一樣得到 (2)。

**第三條（消去時間）**：有些題沒給也不要時間。由 $(1)$ 解 $t=\dfrac{v-v_0}{a}$，代進 $(\ast)$：
$$\Delta x = \frac{v_0+v}{2}\cdot\frac{v-v_0}{a} = \frac{v^2-v_0^2}{2a}\ \Rightarrow\ v^2 = v_0^2 + 2a\Delta x.\tag{3}$$
**第三條完全是前兩條的代數結果，不是獨立的新事實。**

### 所以要不要背？
與其死背四條式子，不如記**一張圖 + 兩個動作**：
- 圖：等加速度的 $v\text{–}t$ 是直線。
- 動作一：**斜率＝加速度** → 得 (1)。
- 動作二：**面積（梯形）＝位移** → 得 (2)、($\ast$)。
- (3) 臨時由 (1) 代 (2) 推，或記成「沒有 $t$ 的那條」。

這樣即使一時忘了公式，也能當場重建，而且不會把符號記反。

> [!warning] 常見迷思
> - **把這些公式用在「非等加速度」上**——三式的唯一前提是「$a$ 為定值」。$a$ 會變（如有空氣阻力、變力）時通通失效。
> - $\Delta x$ 與 $v$ **忘記帶正負號**：先定正方向，減速時 $a$ 取負、反向運動時速度取負，公式才會自洽。
> - 「平均速度 $=\dfrac{v_0+v}{2}$」拿去用在**不是等加速度**的情形——這條 ($\ast$) 同樣只在等加速度成立。
> - 把第三條 $v^2=v_0^2+2a\Delta x$ 當成獨立公式硬記，反而容易把 $2a\Delta x$ 的符號記錯；知道它是推出來的就穩了。

## 回到課堂
- **一句話講法**：「這三條不是天上掉下來要你背的，是同一張 $v\text{–}t$ 直線圖的『斜率』和『面積』，再加一次代入。」
- **板書順序**：先畫 $v\text{–}t$ 直線 → 標斜率得 (1) → 標梯形面積得 (2) → 當場代入消 $t$ 得 (3)。讓學生看著圖「長出」公式。
- **好用提醒**：教學生用「缺哪個量」選公式（缺位移用 (1)、缺末速用 (2)、缺時間用 (3)），比硬記更實用。
- **預期學生會追問**：「考試來不及推怎麼辦？」→「熟練後 (1)(2) 會自動記住，(3) 真忘了花十秒代一下就有；重點是你永遠知道它對不對、適不適用。」
