# 靜力平衡為什麼要「兩個」條件，合力為零還不夠嗎？

> [!question] 課本沒講清楚的地方
> 之前學質點平衡，只要「合力 = 0」一條件就搞定。到了剛體，課本突然說要**兩個**條件：合力 = 0 **而且** 合力矩 = 0。為什麼多出一條？「合力為零」明明代表物體不會被推走了，難道還會發生什麼事？

## 一句話定義

合力為零保證物體**不會平移（不被整體推走）**；但有大小的物體還可能**原地打轉**。要它「既不移、也不轉」，必須再加一條「合力矩為零」。所以剛體平衡要**兩個**獨立條件。

## 細一點：用一支鉛筆親手試

把一支鉛筆平放桌上，用兩根手指在**兩端**、往**相反方向**各推一樣大的力（一個往上、一個往下，如下示意）：

```
        ↑ F
   ━━━━━━━━━━━━   ← 鉛筆
 F ↓
```

- 兩力大小相等、方向相反 → **合力 = 0**。鉛筆的重心確實不會被推走。
- 但這兩個力**不在同一條線上**（一個作用在左端、一個在右端），它們形成一個「**力偶（couple）**」，會讓鉛筆**繞中心打轉**！

這就證明了：**合力為零，物體照樣可以轉起來**。光靠「合力 = 0」管不住轉動。

> [!note] 為什麼質點不需要第二條件？
> 質點是「沒有大小的點」，所有的力都作用在**同一個點**上，根本沒有「不同作用點」造成的轉動問題——力臂全是零，力矩自動為零。所以質點平衡只剩「合力 = 0」一條。
> 一旦物體有了大小、力作用在不同位置，「轉不轉」就成了一個獨立的問題，必須另外用「合力矩 = 0」來管。

## 兩個條件各管什麼

> [!note] 剛體平衡的兩條件
> $$\sum \vec F = 0 \quad\Longrightarrow\quad \text{不平移（質心加速度為零）}$$
> $$\sum \tau = 0 \quad\Longrightarrow\quad \text{不轉動（角加速度為零）}$$
> 兩者是**互相獨立**的：可以「不移卻在轉」（鉛筆力偶），也可以「在移卻不轉」（被一個過重心的力推著直直加速）。要靜止不動，兩條都得滿足。

這其實對應牛頓定律在剛體上的兩種版本：合力決定**質心的平移**（$\sum\vec F = M\vec a_{\text{cm}}$），合力矩決定**繞質心的轉動**（$\sum\tau = I\alpha$，轉動慣量 $I$ 與角加速度 $\alpha$ 到大學才細談）。平衡就是這兩個加速度都為零。

> [!warning] 常見迷思
> - 「合力為零 = 平衡」只對**質點**成立，對剛體不夠。
> - 取力矩的「支點」**可以任意選**。很多學生以為一定要對某個固定軸取矩——其實只要物體平衡，對**任何一點**取矩都會是零。聰明的選法是選在「未知力最多」的點，讓那些力的力臂歸零、方程式變簡單。
> - 「力偶的合力是零，所以沒有效果」——錯。力偶合力雖為零，卻有淨力矩，是純粹的「轉動源」（方向盤、螺絲起子轉動都是力偶）。

## 回到課堂
- **一句話講法**：合力為零只保證「不被推走」，還要合力矩為零才保證「不打轉」。剛體有大小，平移和轉動是兩件事。
- **演示**：當場用鉛筆（或尺）做「兩端反向推」的力偶實驗，讓學生親眼看到「合力為零卻會轉」。
- **板書技巧**：示範「選對支點」——刻意把支點選在未知反力上，讓它的力臂為零，學生會驚訝原來方程式可以這麼短。
- **預期學生會追問**：
  - 「對不同的點取矩，答案會不會不一樣？」→ 物體真的平衡時，對任一點取矩都是零，答案一致；選不同點只是讓計算難易不同。
  - 「那為什麼以前質點不用管力矩？」→ 因為質點沒有大小，所有力同一作用點，力臂全為零。
